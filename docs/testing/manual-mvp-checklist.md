# DevHub MVP 手动验收清单

## 设置

- [x] 首次运行时会自动加载或创建 `settings.json`。
- [x] 修改主题、字体、布局后会校验 JSON 配置。
- [ ] 如果把不允许的敏感字段直接写入 `settings.json`，保存时会被拒绝。
- [ ] 复制 `settings.json` 和 `keymap.json` 可以恢复非敏感工作环境。

## 连接面板

- [ ] 默认存在本地终端连接。
- [ ] 点击添加连接会打开弹窗，而不是跳转页面。
- [ ] 双击连接项会打开对应终端标签。
- [ ] 同一个连接支持通过“新标签连接”打开多个独立终端标签。
- [ ] 连接项右键菜单包含：连接、新标签连接、SFTP、编辑、复制。
- [ ] 复制连接会打开添加连接弹窗，并填入原连接信息。

## SSH

- [ ] 密码登录可用。
- [ ] 私钥登录可用。
- [ ] 带口令的私钥登录可用。
- [ ] 远程终端中的 `sudo` 提示可以正常交互输入。
- [ ] 关闭终端标签后会释放会话。
- [ ] 同时打开 20 个终端标签时界面仍保持响应。
- [ ] 终端会随窗口大小和面板开关重新适配宽度。
- [ ] 切换主题不会重连终端。
- [ ] 切换主题后终端不出现额外截断文本。
- [ ] 新开远程终端不会偶发只显示 `Connected.` 而没有远端 prompt。
- [ ] 终端区域右键菜单包含：复制、粘贴、清屏。
- [ ] 第一次粘贴不会弹出浏览器剪贴板权限提示。
- [ ] 粘贴后删除内容不会出现 `[devhub] io error: transport read`。
- [ ] 粘贴、复制、清屏后焦点仍回到终端，可以继续输入。
- [ ] 清屏只清理终端显示，不关闭连接。

## 工作区标签和右键菜单

- [ ] 初次打开应用时工作区没有默认标签。
- [ ] 打开终端后标签标题为连接名称。
- [ ] 设置页以标签形式打开，可以关闭。
- [ ] 关闭终端标签会关闭对应后端会话。
- [ ] 工作区标签右键菜单包含：关闭、关闭其他、关闭左侧、关闭右侧。
- [ ] 空白工作区右键菜单包含：打开设置、显示连接面板。
- [ ] 未定义右键菜单的区域不会显示浏览器默认菜单。

## SFTP

- [ ] 打开 SFTP 标签时会创建后端 SFTP 会话。
- [ ] 关闭 SFTP 标签时会关闭对应 SFTP 会话。
- [ ] 可以浏览有权限访问的目录。
- [ ] 双击文件夹可以进入该文件夹。
- [ ] 地址栏输入远程路径并回车后可以跳转。
- [ ] 后退按钮可以回到上一个目录。
- [ ] 前进按钮可以回到后退前的目录。
- [ ] 刷新按钮会重新加载当前目录，不改变前进和后退历史。
- [ ] 地址栏输入 `~` 或 `~/` 可以跳转到远端 home 目录。
- [ ] 文件列表显示名称、类型、大小和修改时间。
- [ ] 修改时间按 `yyyy-MM-dd HH:mm:ss` 显示。
- [ ] 点击名称、大小或修改时间可以排序。
- [ ] 文件大小显示方式可以通过设置在字节和自动单位之间切换。
- [ ] 右键文件列表区域包含当前目录操作：刷新、新建文件、新建文件夹、上传文件、上传文件夹。
- [ ] 右键文件夹包含：下载、重命名、复制路径、删除、压缩。
- [ ] 右键文件包含：下载、重命名、复制路径、删除、压缩。
- [ ] 文件列表可以多选条目。
- [ ] 多选条目后右键可以批量下载、删除、压缩和复制路径。
- [ ] 多选压缩会把选中项压缩到同一个 `.tar.gz` 包中，而不是每个条目各自压缩。
- [ ] `.tar.gz` / `.tgz` 文件右键包含解压缩。
- [ ] 可以上传小文件。
- [ ] 可以下载小文件。
- [ ] 可以上传文件夹。
- [ ] 可以下载文件夹。
- [ ] 可以拖拽本地文件到文件列表上传。
- [ ] 可以拖拽本地文件夹到文件列表上传。
- [ ] 上传服务器已有同名文件或文件夹时，会先提示是否覆盖。
- [ ] 上传和下载任务会显示在传输队列中。
- [ ] 传输任务区域高度固定，任务很多时可以滚动查看。
- [ ] 传输中的任务会显示进度百分比。
- [ ] 可以取消进行中的上传或下载任务，任务状态会显示为已取消。
- [ ] 上传或下载失败时，传输队列会显示错误信息。
- [ ] 关闭 SFTP 标签时会取消该标签未完成的传输任务。
- [ ] 可以新建空文件。
- [ ] 可以新建文件夹。
- [ ] 可以重命名文件或目录。
- [ ] 可以删除文件或目录。
- [ ] 删除文件或目录前会有确认。
- [ ] 删除文件夹时确认文案会标明“文件夹”。
- [ ] 可以复制远程路径到剪贴板。
- [ ] 可以打开小于 5MB 的文本类远程文件。
- [ ] 常见 Linux 配置文件名即使没有扩展名也可以打开，例如 `profile`、`bashrc`。
- [ ] 文件编辑弹窗可以拖动、调整大小，并记住上次位置和尺寸。
- [ ] 保存远程文本文件后会刷新文件信息。
- [ ] 没有权限时会显示错误，不会尝试 sudo 提权写入。
- [ ] 同时运行 3 个传输任务时，不会阻塞终端输入。

## MVP 已知限制

- 暂不支持跳板机。
- 暂不支持 SSH agent。
- 暂不支持 SSH tunnel。
- 暂不支持 SFTP sudo 写入。
- 暂不支持传输任务暂停和恢复。
- 暂不支持完整数据库和 Redis 管理。
