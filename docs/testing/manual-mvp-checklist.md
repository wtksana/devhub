# DevHub MVP 手动验收清单

## 设置

- [x] 首次运行时会自动加载或创建 `settings.json`。
- [x] 修改主题、字体、布局后会校验 JSON 配置。
- [x] 如果把不允许的敏感字段直接写入 `settings.json`，保存时会被拒绝。
- [x] 复制 `settings.json` 和 `keymap.json` 可以恢复非敏感工作环境。
- [x] 设置页可以在跟随系统、简体中文和 English 之间切换界面语言。
- [x] 语言设置写入 `settings.json` 的 `appearance.language`，新安装默认值为 `system`。

## 连接面板

- [x] 默认存在本地终端连接。
- [x] 点击添加连接会打开弹窗，而不是跳转页面。
- [x] 添加连接支持密码认证和私钥认证。
- [x] 私钥认证支持填写私钥路径和私钥口令。
- [x] 私钥路径支持通过文件选择器填入。
- [x] 添加连接支持选择 Redis 类型。
- [x] Redis 连接支持填写主机、端口、数据库编号和密码。
- [x] Redis 密码会保存到 `settings.json`。
- [x] Redis 连接项双击会打开 Redis 工作区标签，不会打开终端。
- [x] Redis 连接项右键菜单包含“测试连接”，不显示 SSH 终端和 SFTP 操作。
- [x] Redis 工作区支持选择数据库编号。
- [x] Redis 工作区支持输入关键字模糊匹配后刷新扫描。
- [x] Redis 工作区支持设置 key 分隔符，默认 `:`，已加载 key 会按分隔符展示为目录树。
- [x] Redis 工作区显示当前 DB 的 key 总数和本次已加载数量。
- [x] Redis 工作区一次加载数量默认 5000，可在工具栏修改。
- [x] Redis key 列表显示 key 名称、类型和 TTL。
- [x] Redis 工作区支持新建 `string`、`hash`、`list`、`set` 和 `zset` 类型 key。
- [x] 新建 Redis key 时不会覆盖已有 key。
- [x] 双击 Redis key 会打开详情弹窗。
- [x] Redis key 详情支持查看 `string`、`hash`、`list`、`set` 和 `zset`。
- [x] Redis key 详情读取有上限：`string` 默认最多 5MB，集合类默认最多 500 条。
- [x] Redis key 详情弹窗可以按 ESC 关闭。
- [x] Redis `string` 类型可以在详情弹窗中编辑并保存。
- [x] Redis `hash` 支持新增字段、编辑字段值和删除字段。
- [x] Redis `list` 支持编辑指定下标元素、尾部追加元素和删除指定下标元素。
- [x] Redis `set` 支持新增成员和删除成员。
- [x] Redis `zset` 支持新增成员、编辑成员分数和删除成员。
- [x] Redis key 可以在详情弹窗中设置 TTL 和移除 TTL。
- [x] Redis key 可以在详情弹窗中删除，删除前会二次确认。
- [x] Redis key 可以在详情弹窗中重命名，目标 key 已存在时不会覆盖。
- [x] Redis key 列表行右键菜单包含：编辑、重命名、删除；删除前会二次确认。
- [x] Redis key 列表支持多选。
- [x] Redis key 列表右键菜单支持批量删除，删除前会二次确认。
- [x] Redis key 列表右键菜单支持批量设置 TTL。
- [x] Redis key 列表右键菜单支持批量移除 TTL。
- [x] 连接支持填写分组，分组字段可以输入新分组或选择已有分组。
- [x] 本地终端固定显示在连接面板顶部，不进入任何分组。
- [x] 连接列表按分组展示连接，未填写分组的连接显示在“未分组”中。
- [x] 连接项右键菜单包含“移动到分组”，可以移动到已有分组或“未分组”。
- [x] 连接面板列表空白处右键菜单包含：添加连接、添加分组、分组排序、连接排序。
- [x] 分组排序和连接排序二级菜单包含：按名称、按最后连接时间、按连接最多次数。
- [x] 双击连接项会打开对应终端标签。
- [x] 同一个连接支持通过“新标签连接”打开多个独立终端标签。
- [x] 连接项右键菜单包含：连接、新标签连接、SFTP、编辑、复制。
- [x] 复制连接会打开添加连接弹窗，并填入原连接信息。

## SSH

- [x] 密码登录可用。
- [x] 私钥登录可用。
- [ ] 带口令的私钥登录可用。
- [ ] 远程终端中的 `sudo` 提示可以正常交互输入。
- [x] 关闭终端标签后会释放会话。
- [x] 同时打开 20 个终端标签时界面仍保持响应。
- [x] 终端会随窗口大小和面板开关重新适配宽度。
- [x] 切换主题不会重连终端。
- [x] 切换主题后终端不出现额外截断文本。
- [x] 新开远程终端不会偶发只显示 `Connected.` 而没有远端 prompt。
- [x] 终端区域右键菜单包含：复制、粘贴、清屏。
- [x] 第一次粘贴不会弹出浏览器剪贴板权限提示。
- [x] 粘贴后删除内容不会出现 `[devhub] io error: transport read`。
- [x] 粘贴、复制、清屏后焦点仍回到终端，可以继续输入。
- [x] 清屏只清理终端显示，不关闭连接。

## 工作区标签和右键菜单

- [x] 初次打开应用时工作区没有默认标签。
- [x] 打开终端后标签标题为连接名称。
- [x] 设置页以标签形式打开，可以关闭。
- [x] 关闭终端标签会关闭对应后端会话。
- [x] 工作区标签右键菜单包含：关闭、关闭其他、关闭左侧、关闭右侧。
- [x] 标签栏标签较多时可以横向滚动，不压缩标签宽度。
- [x] 标签标题完整显示，不使用省略号。
- [x] 鼠标滚轮可以横向滚动标签栏。
- [x] 标签栏不显示滚动条。
- [x] 标签栏空白区域保留底部分隔线，激活标签下方不显示分隔线。
- [x] 空白工作区右键菜单包含：打开设置、显示连接面板。
- [x] 未定义右键菜单的区域不会显示浏览器默认菜单。

## SFTP

- [x] 打开 SFTP 标签时会创建后端 SFTP 会话。
- [x] 关闭 SFTP 标签时会关闭对应 SFTP 会话。
- [x] 可以浏览有权限访问的目录。
- [x] 双击文件夹可以进入该文件夹。
- [x] 地址栏输入远程路径并回车后可以跳转。
- [x] 后退按钮可以回到上一个目录。
- [x] 前进按钮可以回到后退前的目录。
- [x] 刷新按钮会重新加载当前目录，不改变前进和后退历史。
- [x] 地址栏输入 `~` 或 `~/` 可以跳转到远端 home 目录。
- [x] 文件列表显示名称、类型、大小和修改时间。
- [x] 修改时间按 `yyyy-MM-dd HH:mm:ss` 显示。
- [x] 点击名称、大小或修改时间可以排序。
- [x] 文件大小显示方式可以通过设置在字节和自动单位之间切换。
- [x] 右键文件列表区域包含当前目录操作：刷新、新建文件、新建文件夹、上传文件、上传文件夹。
- [x] 右键文件夹包含：下载、重命名、复制路径、删除、压缩。
- [x] 右键文件包含：下载、重命名、复制路径、删除、压缩。
- [x] 文件列表可以多选条目。
- [x] 多选条目后右键可以批量下载、删除、压缩和复制路径。
- [x] 多选压缩会把选中项压缩到同一个 `.tar.gz` 包中，而不是每个条目各自压缩。
- [x] `.tar.gz` / `.tgz` 文件右键包含解压缩。
- [x] 可以上传小文件。
- [x] 可以下载小文件。
- [x] 可以上传文件夹。
- [x] 可以下载文件夹。
- [x] 可以拖拽本地文件到文件列表上传。
- [x] 可以拖拽本地文件夹到文件列表上传。
- [x] 上传服务器已有同名文件或文件夹时，会先提示是否覆盖。
- [x] 上传和下载任务会显示在传输队列中。
- [x] 传输任务区域高度固定，任务很多时可以滚动查看。
- [x] 传输中的任务会显示进度百分比。
- [x] 可以取消进行中的上传或下载任务，任务状态会显示为已取消。
- [x] 上传或下载失败时，传输队列会显示错误信息。
- [x] 关闭 SFTP 标签时会取消该标签未完成的传输任务。
- [x] 可以新建空文件。
- [x] 可以新建文件夹。
- [x] 可以重命名文件或目录。
- [x] 可以删除文件或目录。
- [x] 删除文件或目录前会有确认。
- [x] 删除文件夹时确认文案会标明“文件夹”。
- [x] 可以复制远程路径到剪贴板。
- [x] 可以打开小于 5MB 的文本类远程文件。
- [x] 常见 Linux 配置文件名即使没有扩展名也可以打开，例如 `profile`、`bashrc`。
- [x] 文件编辑弹窗可以拖动、调整大小，并记住上次位置和尺寸。
- [x] 保存远程文本文件后会刷新文件信息。
- [ ] 没有权限时会显示错误，不会尝试 sudo 提权写入。
- [x] 同时运行 3 个传输任务时，不会阻塞终端输入。

## MVP 已知限制

- 暂不支持跳板机。
- 暂不支持 SSH agent。
- 暂不支持 SSH tunnel。
- 暂不支持 SFTP sudo 写入。
- 暂不支持传输任务暂停和恢复。
- 暂不支持完整数据库管理。
- 暂不支持 Redis 发布订阅和监控面板。
