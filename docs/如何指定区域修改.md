如何指定区域修改：

**整体结构**
- `应用外壳`：整个 DevHub 主窗口。
- `顶部栏` / `命令栏`：最上面一整条区域，包含 DevHub 标题、打开设置、窗口按钮。
- `窗口控制区`：右上角最小化、最大化、关闭按钮。
- `主工作区`：顶部栏和底部状态栏之间的主体区域。
- `底部状态栏`：最底部一条，包含连接面板显示/隐藏按钮和后续状态入口。

**左侧区域**
- `连接面板` / `连接列表`：左侧显示 SSH、数据库、Redis 连接的区域。
- `连接面板头部`：连接列表顶部标题和操作按钮区域。
- `连接项`：单个连接配置条目。
- `连接空状态`：暂无连接时显示的提示区域。

**中间区域**
- `工作区`：中间主要内容区域。
- `工作区标签栏`：中间顶部的 `终端 / SFTP / 设置` 标签。
- `终端工作区`：打开终端时的中间区域。
- `终端标签页`：具体一个 SSH 终端会话区域。
- `SFTP 工作区`：打开 SFTP 时的文件管理区域。
- `设置页`：打开设置时的中间设置编辑区域。

**设置页**
- `设置侧边栏`：设置页左侧分类导航。
- `设置搜索框`：设置侧边栏顶部搜索框。
- `设置分类导航`：通用、外观、布局、连接、快捷键、settings.json。
- `设置内容区`：设置页右侧可滚动内容。
- `设置分组`：例如“外观”“布局”“连接”这些块。
- `设置项`：每一行具体配置，比如主题、界面字体、终端字号。
- `设置项控件`：设置项右侧的输入框、下拉框、开关。
- `settings.json 编辑区`：JSON 直接编辑区域。
- `设置页滚动条`：设置内容区右侧滚动条。

**右键菜单**
- `连接项右键菜单`：连接列表里单个连接项的菜单。
- `标签右键菜单`：工作区标签栏里单个标签的菜单。
- `空白工作区右键菜单`：工作区没有内容或空白处的菜单。
- `终端右键菜单`：终端内容区域里的菜单。
- `浏览器默认右键菜单`：WebView 自带菜单。当前约定是未定义菜单的区域不显示它。

**常用说法示例**
可以直接说：
- “把顶部栏高度调小一点”
- “设置页滚动条颜色再暗一点”
- “连接面板和工作区之间的分割线太明显”
- “设置项控件太靠右”
- “窗口控制区关闭按钮 hover 再融合一点”
- “底部状态栏按钮改得更像 Zed”
- “终端右键菜单加一个重新连接”
- “标签右键菜单里的关闭右侧没生效”
